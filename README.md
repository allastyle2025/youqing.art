# 有情 - 素人共创生态舞作网站

这是一个为"有情"素人共创生态舞作项目打造的艺术感十足的展示网站，旨在呈现素人舞蹈的魅力、分享舞者故事并提供视频画廊展示功能。

## 项目简介

"有情"是永·舞团在2024年发起的素人共创生态舞作项目，核心理念是"人人皆可舞，以身体拥抱自然"。项目汇聚了来自各行各业的素人舞者，共同创作、排演，探索身体与自然、艺术与生命的深层连接。

## 网站特点

- **艺术感设计**：采用简约而富有艺术感的界面设计，融入自然元素和舞蹈灵感
- **视频画廊**：展示演出、排练和纪录片视频，支持分类筛选
- **舞者故事**：讲述来自各行各业的素人舞者的舞蹈之旅和感悟
- **响应式布局**：完美适配桌面、平板和移动设备
- **互动体验**：丰富的动画效果和交互体验，提升用户浏览体验
- **演出日程**：展示未来的演出安排和活动信息

## 技术栈

- HTML5
- CSS3 (Tailwind CSS v3)
- JavaScript
- Font Awesome 图标库
- Chart.js (数据可视化)
- Google Fonts (字体资源)

## 如何运行

1. 确保您的设备上安装了Web服务器环境（如XAMPP、WAMP等）
2. 将项目文件放置在Web服务器的根目录下
3. 启动Web服务器
4. 在浏览器中访问 `http://localhost/youqing/` 或相应的项目路径

## 目录结构

```
youqing/
├── index.html     # 主页面
├── README.md      # 项目说明文档
└── .gitignore     # Git忽略文件配置
```

## 自定义指南

### 修改内容
- 如需更新舞者信息、视频内容或演出日程，请直接编辑`index.html`文件中对应的部分
- 可以替换示例图片为实际项目照片，保持相同的文件路径或更新路径引用
- 可根据需要调整配色方案（在`<script>`标签内的Tailwind配置中）

### 添加功能
- 可以扩展视频画廊，添加更多分类和视频内容
- 可考虑添加在线报名表单，实现演出预约功能
- 可以集成社交媒体分享功能，提升项目影响力

## 注意事项

- 网站中的视频链接目前使用示例链接，实际项目中请替换为真实视频URL
- 为保证最佳体验，请确保所有图片和视频资源已优化
- 在生产环境中，建议压缩CSS和JavaScript文件以提高加载速度
